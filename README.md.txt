# Autonomous Mobile Robotics – Homework Submissions

This repository contains multiple homework assignments completed for the
Autonomous Mobile Robotics course.

## Contents

### HW3 – Kinematics & Dynamic Control
- 3-wheel omni robot kinematics
- PI wheel-speed control
- Sliding Mode Control (SMC)
- Phase portraits and sliding surface analysis

### HW6 – Control Barrier Functions (CBF)
- Safe navigation around a pedestrian
- Barrier function formulation
- Animated trajectory

### HW7 – Potential Field Navigation
- Attractive + repulsive potential fields
- Successful navigation case
- Failure due to local minimum
- Python-based animation

Animations and plots are included where applicable.
